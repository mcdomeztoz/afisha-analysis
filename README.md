# Анализ лояльности пользователей Яндекс Афиши

## Описание
Исследовательский анализ данных о покупках билетов на платформе Яндекс Афиша.
Цель — выявить факторы, влияющие на повторные заказы пользователей.

## Этапы работы
1. Выгрузка данных из PostgreSQL через SQLAlchemy
2. Предобработка данных (конвертация валют, обработка выбросов)
3. Создание профиля пользователя
4. Исследовательский анализ данных
5. Корреляционный анализ (phi_k)
6. Выводы и рекомендации

## Стек
- Python, Pandas, NumPy
- Seaborn, Matplotlib
- SQLAlchemy, PostgreSQL
- phik
- os, dotenv

## Установка
1. Клонируй репозиторий
2. Установи зависимости:
   pip install -r requirements.txt
3. Создай файл .env с параметрами подключения к БД (см. пример ниже)

## Пример .env
DB_HOST=твой_хост
DB_NAME=имя_базы
DB_PORT=6432
DB_USER=твой_пользователь
DB_PASS=твой_пароль
